# 项目介绍
该项目用于将知乎的收藏夹（公开）导出为 markdown 格式的文件，供学习和研究使用。

# 安装
首先安装python

然后安装依赖
```bash
pip install -r requirements.txt
```
# 使用
## 终端
首先，请找到您想导出的收藏夹的 url
然后，在终端（cmd）界面下输入
```bash
python main.py [收藏夹的url]
```

## 文件存放目录
等待下载完成
- markdown 文件将保存在当前操作系统的"**下载**"目录下的"**剪藏**"文件夹下，
- 文章中的图片将保存在剪藏文件夹中的"**assets**"文件夹中
